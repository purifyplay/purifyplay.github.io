Decrypt Link Lock URLs

Esta aplicación es para descifrar Link Lock URL sin redirección automática. Esto es útil si no confía en la fuente de una URL cifrada. También es útil si la URL usa un protocolo bloqueado como javascript:, por ejemplo.

Esta página también es útil si cree que ha recibido un enlace bloqueado, pero utiliza otro dominio, en lugar de gitmaster-user.github.io. Esto puede hacerse como un medio para evadir la censura.
