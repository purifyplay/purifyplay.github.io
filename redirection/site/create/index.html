Bloqueo de enlaces: enlaces protegidos con contraseña

Link Lock es una herramienta para agregar una contraseña a un enlace; en otras palabras, para cifrar y descifrar URL. Cuando un usuario visita una URL cifrada, se le solicitará una contraseña. Si la contraseña es correcta, Link Lock los envía al sitio web oculto. De lo contrario, se muestra un error. Los usuarios también pueden agregar pistas para recordarles la contraseña.

Cada URL encriptada se almacena completamente dentro del enlace generado por esta aplicación. Como resultado, los usuarios controlan todos los datos que crean con Link Lock. Nunca se almacena nada en un servidor, y no hay cookies, seguimiento o registros. Ver en GitHub para obtener más información, incluidas las versiones traducidas. on GitHub for more information, including translated versions.

Link Lock tiene muchos usos, por ejemplo:

  • Agregue una contraseña a los enlaces compartidos de Dropbox o Google Drive
  • Implementar CAPTCHA simples
  • Cifrar páginas completas (a través de páginas URL) - Contraseña: urlpage5 (via URL Pages) - Password: urlpage5
  • Publicar enlaces privados en sitios web públicos
  • Comparta torrents protegidos con contraseña y enlaces magnéticos - Password: torrenting_is-legal!
